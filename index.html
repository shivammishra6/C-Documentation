Introduction

The programming language C was developed by Dennis Ritchie between 1972 and 1973. It was inspired by the language B. The place where the language was built is known as Bell Laboratories. It is a language used to build system software, application software and embedded systems.

C has inspired many languages, for example look at C++. It is a language which is based on C and adds features such as OOP(Object Oriented Programming). Furthermore you can find similarity in Java,C# etc.

C is a middle level language. It does not support garbage collector. In the modern era C is not used extensively but is still effective.

Basic Boilerplate

The basic boiler plate code for the C language is

#include stdio.h

int main()
{
printf("Hello world!");
return 0;
}

In the above code you can see the include keyword which is used to include library files in C. In this case we have included stdio.h which stands for 'standart input/output'. It supports basic functionalities like printf(),scanf().

scanf example:

scanf("%d",&a);
Datatypes

The datatypes in C language are divided in 3 categories:

  • Primitive data types

    • The primitive data type or built in data type has 4 components:

    1. int
    2. float
    3. char
    4. boolean
  • Derived data types

    • The derived data types are data types which are the outcome of the process done on the primitive data types.

    Example of such data types include:

    1. Stack
    2. Queue
    3. Linked list
    4. Tree
  • User defined data types

    • These kind of data types are made by the user and for the user.

Some common examples include:

  • Enum
  • Typedef
String

String in the c language is also known as char array. C language does not provide the String as an datatype, rather you will have to use a code like this to declare a string:

char name[20];

The above code defines a variable name of 20 characters as a string in the c language

To declare an array of string, you can refer the below example:

char name[5][20];

The above will declare 5 strings of 20 characters.

Structure

Structures in C provide you with the feature of using multiple datatypes with the use of only one variable.

The typical Syntax of structure looks like:

struct student
{
int rollno;
char name[20];
}

Explanation of the above code tokens:

  • The struct keyword is used to declare structure
  • student is the name of the structure
  • the code block starts with curly braces and end with curly braces
  • The variables declared in the braces are called structure members.
UDF

The term UDF stands for 'User Defined Function'. This help us in making utility functions for our main function to use.

There are four types of UDF:

  • Function with no arguments and no return value.
  • Function with no arguments and a return value.
  • Function with arguments and no return value.
  • Function with arguments and with return value.
